## English

This repository is a Seedance 2.5 prompt library and case index maintained by HiAPI for creators, developers, and AI agents.

It contains two types of material:

1. **HiAPI-owned material**: repository curation, README structure, original launch templates, scripts, JSON schema, explanatory copy, and SEO organization.
2. **Third-party source index**: video URLs, preview image URLs, reference material URLs, case categories, and prompt-theme summaries in `data/official-cases.json`, compiled from the public Seedance 2.5 promotion page: <https://ark.volcengine.com/promotion?modelName=seedance-2-5>.

The source page states that its materials are generated by the Seedream and Seedance model families. This repository indexes, categorizes, and summarizes those cases for learning and discovery. HiAPI does not claim ownership of the third-party source videos, images, reference materials, original prompts, brand names, model names, or platform names.

To avoid large-scale republication of third-party page text, this repository does not mirror the full long prompts from the source page. To copy the original prompt verbatim, open the source page. `data/official-cases.json` keeps prompt visibility status, theme summaries, video links, and reference material links so users can locate and study each case.

Seedance is a ByteDance model. This repository is an independent prompt library and is not affiliated with, sponsored by, or endorsed by ByteDance, Volcano Engine, or their affiliates.
